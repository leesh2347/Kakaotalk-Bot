사용 방법
1. 구글 Gemini api key를 발급받은 후, server.py의 api key 선언 부분에 입력합니다.
2. data 폴더 내부의 파일은 tool_instructions.txt는 AI에게 따르도록 학습시킬 지침, words.txt는 채팅방 유저들이 자주 사용하는 축약어들을 적어놓은 것입니다. 해당 파일들을 우선 확인해 주세요.
3. 프로젝트 루트 디렉토리에서 콘솔 창을 엽니다.
4. python embedding_index.py를 입력하고 실행시킵니다. 시간이 좀 걸리니 기다리세요.
실행이 완료되면 지침과 축약어 목록의 RAG 인덱스 청크 파일이 data 폴더 내부에 생성됩니다. (예시: faiss_index.idx, faiss_index_meta.pkl)
5. python server.py를 입력하고 실행시킵니다.
6. localhost:8000/ask url에 백엔드 서버가 실행됩니다.
봇을 구동시킬 기기를 서버 실행 장비와 같은 네트워크에 연결시키거나, 외부 접속이 가능하도록 포트포워딩을 해 주세요.
7. 메신저봇(프론트엔드)에서 사용하는 법
111.111.111.111:8000/ask
위 URL에 POST 요청을 보내면 됩니다. body에는 json 형식으로 다음과 같이 내용을 담아야 합니다.
{
  "question":"자연어 질문",
  "use_tools":"true"
}

예시
111.111.111.111:8000/ask POST
{
  "question":"타락파워전사 닉네임 검색해줘",
  "use_tools":"true"
}

응답은 항상 다음 json 형식을 따릅니다.
{
  "정보":"명령어",
  "옵션":"필요 인자. 다수일 경우 컴마(,)로 구분되어 있습니다."
}

예시
{
  "정보":"경험치",
  "옵션":"디벨로이드, 2025-06-01, 2025-06-02"
}
